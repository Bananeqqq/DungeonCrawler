# DungeonCrawler
This is a project of my first game in a Godot engine.

	Movement - WASD/←↓↑→	
	Toggle inv - I/Alt
	Toggle shop - O

	Prev hotbar item - Q
	Next hotbar item - E
	First hotbar item - 1
	Second hotbar item - 2
	Third hotbar item - 3

	Pick-up an item - F
	Use an item from hotbar - X

	Drag & drop - left mouse button
